## üìù Communication Channels

### Daily Standup
- **When**: Daily, 15 minutes
- **Who**: All team members
- **Format**: What did I do yesterday? What am I doing today? Any blockers?

### Sprint Planning
- **When**: Start of each sprint
- **Who**: All team members
- **Duration**: 2-3 hours

### Sprint Review
- **When**: End of each sprint
- **Who**: All team members + stakeholders
- **Duration**: 1-2 hours

### Retrospective
- **When**: End of each sprint
- **Who**: All team members
- **Duration**: 1 hour

## üìã RACI Matrix Example

| Activity | PM | Frontend/UI/UX | Full-Stack | Data | DevOps | QA |
|----------|----|----------------|-----------|------|--------|----|
| Sprint Planning | A | R | R | R | R | R |
| Feature Development | I | C | A | C | I | C |
| Code Review | I | R | R | R | R | R |
| Testing | I | C | C | C | I | A |
| Deployment | I | C | C | I | A | C |
| Data Pipeline | I | I | C | A | C | C |
| Design | I | A | C | I | I | I |

**Legend:**
- **A** = Accountable (ultimately responsible)
- **R** = Responsible (does the work)
- **C** = Consulted (provides input)
- **I** = Informed (kept in the loop)
